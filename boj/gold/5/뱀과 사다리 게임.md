### 문제 이해
주사위 1-6 / 10x10 보드판
주사위 굴려 나온 수만큼 이동
도착한 칸이 사다리면 사다리 타고 이동
뱀이 있으면 뱀따라서 내려감
사다리 - 원래 칸보다 큼 / 뱀 - 원래 칸보다 작음
100번 칸에 도착하기 위해 굴려야하는 주사위의 최솟값
입력
사다리 갯수 N, 뱀의 수 M
N개의 사다리 정보 x, y
M개의 뱀 정보 u, v
### 추상화
DFS/BFS를 이용한 탐색?
	1-6 여섯가지 방향?을 순회하면서 탐색

### 계획
사다리/뱀 초기화
	dict에 x,y u,v 저장
visited 초기화(길이 101, 0번째는 무시)
방향 1~6
1부터 큐에 넣고 탐색 시작
큐에서 값을 꺼냄
방향 여섯가지에 대해서
	사다리 / 뱀 dict 에 다음 위치 조회
		있으면 해당 값을 방문처리하고 큐에 인입
		없으면 다음 위치 방문 처리하고 큐에 인입
		방문 처리시 visited에 현재 스텝 + 1 저장
visited[100] 반환
### 코드
```python
import sys

input = sys.stdin.readline
from collections import deque


def solve(info: dict[int, int]) -> int:
    dist = [-1] * 101
    dist[1] = 0
    steps = [i for i in range(1, 7)]
    q = deque([1])
    while q:
        cur = q.popleft()
        for step in steps:
            nxt = cur + step
            if nxt in info:
                nxt = info[nxt]
            if nxt <= 100 and dist[nxt] == -1:
                dist[nxt] = dist[cur] + 1
                q.append(nxt)
    return dist[100]


if __name__ == "__main__":
    N, M = map(int, input().split())
    info = {}
    for _ in range(N):
        x, y = map(int, input().split())
        info[x] = y
    for _ in range(M):
        u, v = map(int, input().split())
        info[u] = v
    print(solve(info))

```
