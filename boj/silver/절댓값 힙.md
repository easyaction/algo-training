### 문제 이해
절댓값 힙을 이용하여 다음 연산 구현
정수 x를 넣기(push())
배열에서 절댓값이 가장 작은값 출력 후 제거(pop()?)
절댓값 같은 값이 여러개면 가장 작은 수를 출력 후 제거
연산 갯수 N
0 -> pop
1 <= x < 2^31 -> push

### 추상화
절댓값을 기준으로 heapify를 하되, 원래 부호를 기억해야함
별도로 음수인지 양수인지 저장하면 복구가 가능함

### 계획
push()와 pop()을 구현
push(list)-> list, pop(list) -> int
heapq를 사용하는 절댓값 힙 외에 별도로 절댓값을 key로 하는 dict를 만들고, count를 기록
	길이 2의 리스트를 넣어 count를 [ 음수, 양수 ] 로 기록
	음수 우선 출력
### 코드
```python
import sys
input = sys.stdin.readline
import heapq
    
def solve(heap, abs_cnt, n):

            print(0)
    else:
```
