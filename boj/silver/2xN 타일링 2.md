#### 문제 이해

2 x n 직사각형을 1 x 2, 2 x 1, 2x2 타일로 채우는 방법 가짓수
1 <= n <= 1000

가짓수를 f(n)이라고 할때, 
f(1)=1
f(2)= 3
f(3)=5
f(4)=11
f(5)=22
f(6)=33
#### 추상화
새로운 경우의 수를 이전에 계산한 경우의 수에 추가하여 계산(DP)
f(n)= 2 * f(n-2)+f(n-1) 
n-2에서 2x1 두개, 2x2 한개 추가
n-1에서 1x2 한개 추가

### 계획
n을 입력받는다
계산 결과를 저장할 map 선언
f(1), f(2) 초기화
f(n) 순차적으로 계산
	계산시 10,007 나머지 연산
출력
