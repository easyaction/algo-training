### 문제 이해
양수와 +, -가 주어질때, 괄호를 적절히 쳐서 얻을 수 있는 최소값을 구하라
입력 - 식(0~9, +, -로 구성)
출력 - 숫자
### 추상화
음수가 나오고나서 이후 나오는 +를 모두 괄호로 묶으면 차감이 최대로 됨
입력을 하나씩 읽으면서 최소값이 나오도록 분기(그리디)
### 계획
0~9 -> sum에 합산, 첫번째 이후로는 기존 값 * 10
-기호 -> 괄호 시작 이후 다시 - 나올때까지 별도 값으로 저장. 다시 -나오면 별도값을 차감

### 코드
```python
import sys
input = sys.stdin.readline


def solve(input_string) -> int:
    sum = 0
    minus_sum = 0
    integer = 0
    is_minus = False
    for c in input_string:
        if ord('0') <= ord(c) <= ord('9'):
            integer *= 10
            integer += int(c)
        else:
            if c == '-':
                if is_minus:
                    sum -= minus_sum
                    minus_sum = 0
                else:
                    sum += integer
                is_minus = True
            else:
                if is_minus:
                    minus_sum += integer
                else:
                    sum += integer
            integer = 0
    return sum - (minus_sum+integer) if is_minus else sum + integer

if __name__ == "__main__":
    input_string = input().split()[-1]
    print(solve(input_string))

```
