#### 문제 이해하기
N개의 숫자에서 i번째부터 j번째 숫자 합을 M번 수행
- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- 1 ≤ i ≤ j ≤ N
#### 문제 추상화
m번 수행할때마다 i, j 구간에 접근하여 합을 구하면 O(N * M)이 될 것
n번째까지의 합을 저장해두고, j번째까지의 합 - i-1번째 합을 하면 O(N)에 구할 수 있을 것이다.

#### 계획
N개의 숫자 입력받을때 숫자 합을 계산하여 dp에 저장
M개의 i,j마다 dp[j] - dp[i-1]을 계산
dp[0]=0을 넣어줘야 하겠다.
