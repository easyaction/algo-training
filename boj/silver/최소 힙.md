### 문제 이해
최소 힙을 이용하여 다음 연산 구현
자연수 x를 넣기(push())
배열에서 가장 작은값 출력 후 제거(pop()?)
연산 갯수 N
0 -> pop
1 <= x < 2^31 -> push

### 추상화
말 그대로 최소 힙의 push / pop을 구현?
라이브러리만 사용하면 별도 구현없이 가능할 듯
### 계획
push()와 pop()을 구현
push(list)-> list, pop(list) -> int
### 코드
```python
import sys
input = sys.stdin.readline
import heapq
    
def solve(list, n):
    if n == 0:
        if len(list) > 0:
            print(heapq.heappop(list))
        else:
            print(0)
    else:
        heapq.heappush(list, n)
        

if __name__ == "__main__":
    [N] = input().split()
    list = []
    for _ in range(int(N)):
        n = int(input().split()[-1])
        solve(list, n)

```
