## 새싹
아래와 같은 문자를 출력하는 문제.
```
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
```
string escape에 대한 이해를 필요로 함.
golang에서는 백스페이스로 필요한 escape를 해주면 된다.
백스페이스와 double quote만 해주면 출력은 정상적으로 됨

## 두 수 비교
두 수를 공백단위로 받아서 비교한 결과를 출력

#### 풀이
input을 받아 a/b 두 정수로 할당하고, 비교한 뒤 결과값을 출력
golang에서는 fmt.Scan 류를 이용하면 될듯하다.
scanf를 이용하여 명시적으로 공백으로 두 수를 구분해서 받음

### 단어의 갯수
입력받은 문자열의 단어의 갯수를 세는 문제.
각 단어는 공백으로 구분되고, 문자열의 시작과 끝 공백은 무시한다.

풀이
reader + fscan을 이용하여 평소에 입력을 처리했는데, 공백을 포함한 입력을 받기위해 bufio.Scanner를 활용하였다.
그러나 Scanner의 default buf size가 64 * 1024로 설정되어있기 때문에 최대 길이 1,000,000의 문자열을 처리할 수 없어 에러가 발생하였다.
이를 해결하기 위해서는 Scanner의 maxTokenSize를 수정하거나, bufio.Reader의 ReaderString()의 delim으로 개행을 전달해주면 된다.
ReaderString에서는 collectFragments()에서 delim을 발견할때까지 buffer를 늘려가며 ReadSlice를 한다.
그에 맞춰서 buf가 늘어나기때문에 잘 된다.
```
// MaxScanTokenSize is the maximum size used to buffer a token

// unless the user provides an explicit buffer with [Scanner.Buffer].

// The actual maximum token size may be smaller as the buffer

// may need to include, for instance, a newline.

MaxScanTokenSize = 64 * 1024
```
